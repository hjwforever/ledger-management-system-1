# ledger-management-system
    Front-end project of Ledger Management System]
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

## Docker
```
## pull image
docker pull aruoxi/lms-frontend

## run a container
docker run -p 3000:80 \
-d --name lms-frontend \
aruoxi/lms-frontend
```
